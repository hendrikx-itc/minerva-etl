:mod:`minerva.schemacontextbuilder` --- SchemaContextBuilder Module
===================================================================

The SchemaContextBuilder module can be used to generate all XML element handlers for an XML schema that can be used in combination with the XmlDocHandler. It can use the object structure generated by the XmlSchemaParser module.
